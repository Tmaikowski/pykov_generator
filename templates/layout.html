Markov Chains: A Python Implementation

// About the Plotkov Generator

This is a simple implementation of Markov Chains written in Python using the Flask micro-framework. From a high level, the program scrapes a list of the top 1000 movies (according to IMDB) and creates various Markov Chains of different orders used to generate new, never-before-seen, original plot lines

The Markov Chains were processed before hand and serialized using Pickle so the user doesn't have a long wait time while loading the application for the first time. While imperfect and leaving a lot to be desired from a grammar standpoint, the plot generator produces some interesting, hillarious, and sometimes haunting plot lines. Click the 'Generate Plot' button below and see what you come up with!
